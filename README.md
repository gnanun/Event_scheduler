A beautiful, professional event scheduler with zero dependencies — just open and use.

================================================================================
FEATURES
================================================================================

✅ Mini Calendar         - Visual month view with event indicators
✅ Event List            - Clean, sortable list of all your events
✅ Create Events         - Title, description, date, time, location
✅ Edit Events           - Update any event details
✅ Delete Events         - Remove events you don't need
✅ Search Events         - Filter by title or description
✅ Statistics Dashboard  - Total, upcoming, and completed events
✅ Local Storage         - Events persist after browser close
✅ Responsive Design     - Works on desktop, tablet, and mobile
✅ Modern UI             - Gradient design with smooth animations
✅ Confetti Celebration  - Fun feedback when creating events

================================================================================
INSTALLATION
================================================================================

Option 1 - Direct Download:
  1. Download tempus.html
  2. Double-click to open in browser
  3. Start scheduling!

Option 2 - Clone Repository:
  git clone https://github.com/yourusername/tempus-scheduler.git
  cd tempus-scheduler
  open tempus.html

================================================================================
USAGE GUIDE
================================================================================

Creating an Event:
  1. Click "New Event" button
  2. Fill in title, date, time, description, location
  3. Click "Save" — enjoy the confetti!

Viewing Events by Date:
  - Click any date on the mini calendar
  - Events list automatically filters to that day

Searching Events:
  - Type in the search bar to filter by title or description

Editing an Event:
  - Click the edit icon (✏️) on any event card
  - Update details → click "Save"

Deleting an Event:
  - Click the delete icon (🗑️) on any event card
  - Confirm deletion

DATA PERSISTENCE
================================================================================

All events are stored in your browser's localStorage:

✅ Events survive page refresh
✅ Events survive browser restart
❌ Events do NOT sync across devices
❌ Clearing browser data will remove events
